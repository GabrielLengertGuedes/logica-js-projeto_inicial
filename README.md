# Jogo do Número Secreto

Projeto desenvolvido com **HTML5, CSS3 e JavaScript** para praticar **lógica de programação**, estruturas condicionais, laços de repetição e interação com o usuário no navegador.

O jogo sorteia um número aleatório e o usuário deve tentar adivinhar qual é o número secreto. A cada tentativa, o sistema informa se o número digitado é maior ou menor que o valor correto, até o acerto final.

## Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript

## Conceitos praticados

- Variáveis e operadores
- Estruturas condicionais (`if` / `else`)
- Laço de repetição (`while`)
- Geração de número aleatório com `Math.random()`
- Conversão de valores com `parseInt()`
- Manipulação de entrada e saída com `prompt()`, `alert()` e `console.log()`
- Integração entre HTML, CSS e JavaScript

## Funcionalidades

- Sorteio aleatório de um número secreto
- Escolha de números em um intervalo definido
- Dicas para o usuário informando se o número secreto é maior ou menor
- Contagem de tentativas até o acerto
- Mensagem final personalizada com o total de tentativas
